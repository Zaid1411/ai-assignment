# AI Product Categorization API

This project is a FastAPI backend that generates product categories and environmental impact reports.

## Features
- Product category generation
- SEO tags generation
- Product filters
- Environmental impact calculation

## Tech Stack
- Python
- FastAPI
- Pydantic
- Uvicorn

## Run the project

Install dependencies:

pip install -r requirements.txt

Run the server:

python3 -m uvicorn main:app --reload

Open API documentation:

http://127.0.0.1:8000/docs
